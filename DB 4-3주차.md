## unique 

unique : 중복 제거 코드 

```ruby
 select T.course_id
 from course as T
 where unique ( select R.course_id
                           from section as R
                           where T.course_id= R.course_id 
                                       and R.year = 2017);
```

위와 동일한 표현은 아래와 같음 

```ruby
select course_id
from section
where year = 2017
group by course_id
having count(*) = 1;
```

위 코드는 section 테이블의 primary key가 course_id, sec_id, semester, year 로 설정되어서 반드시 하나만 나옴 

따라서 group by를 해도 하나만 나오기 때문에 unique와 같은 역할
